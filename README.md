# Tailwind_RecipeFood

## Getting Started

Make sure you have latest version of Node environment before clone this repo

### Installing

```
npm install
```

## Running

```
npm run build-css
```

opening with Live server

## Built With

- [TailWind](https://tailwindcss.com/) - The css framework used
- [NPM](https://npmjs.com/) - Dependency Management
